# dotfiles

my dots :)

## Setup

```bash
git clone https://github.com/pitriq/dotfiles.git ~/dotfiles
cd ~/dotfiles
./setup.sh
```

## What's in here?

- Zsh with oh-my-zsh, autosuggestions, and syntax highlighting
- Starship prompt
- Homebrew packages (via Brewfile)
- Git configuration
- Ghostty terminal config
- Custom aliases and functions
